### IDE Hyper chords (bind once per IDE)

| Key | Sends   | Action          |
|-----|---------|-----------------|
| Q   | вҢҳвҢғвҢҘвҮ§+B | Build           |
| W   | вҢҳвҢғвҢҘвҮ§+R | Run             |
| E   | вҢҳвҢғвҢҘвҮ§+T | Run tests       |
| R   | вҢҳвҢғвҢҘвҮ§+D | Debug           |
| T   | вҢҳвҢғвҢҘвҮ§+F | Format          |
| V   | вҢҳвҢғвҢҘвҮ§+G | Go to def       |
| B   | вҢҳвҢғвҢҘвҮ§+U | Find usages     |
| Z   | F9      | Toggle BP       |
| X   | F8      | Step over       |
| C   | F7      | Step into       |

## Practice plan

1. **Day 1вҖ“3**: Just type. Get comfortable with base QWERTY + Backspace/Space/Enter/Cmd on thumbs.
2. **Day 4вҖ“7**: Add NAV. Force yourself to use hjkl instead of arrow keys.
3. **Week 2**: Add SYM. Replace number-row reaches with the numpad.
4. **Week 3**: Add FN. Bind Hyper chords in your IDE.
